# SWRTHY

This repo contains the SWRTHY parent company website and supporting assets.

## Website

The website is a simple static site located in the `docs/` folder so it can be hosted via GitHub Pages.

### Publish to GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Choose **Branch: main** and **Folder: /docs**.
4. Save and wait a few minutes for the site to be deployed.

### Change the site copy

Update `docs/index.html` and `docs/styles.css`.
